# e-commerce-backend

## Description

This application provides backend database functionality for an eCommerce site. The express-based API allows the user to view, create, update and delete products, categories and associated product tags.

## Installation <a name="installation"></a>

This application requires the following packages from npm:
mysql2
express
dotenv
nodemon
sequelize

Create ecommerce db by sourcing schema.sql. 'npm run seed' will seed the database with starter data.

## Usage <a name="usage"></a>

```
> npm start
```
